Day 1 · 第1-10词 · 2026-04-23
📝 今日单词
#1
a /ə/ /eɪ/
Lv.1 art. 一个；任一
例句 Example
I have a book.
我有一本书。
#2
able /ˈeɪbl/
Lv.1 adj. 能够的；有能力的
例句 Example
I want to learn how to use 'able' correctly.
我想学习如何正确使用'able'。
#3
about /əˈbaʊt/
Lv.1 prep. 关于；大约 adv. 大约
例句 Example
Tell me about your day.
告诉我你今天过得怎么样。
#4
above /əˈbʌv/
Lv.1 prep./adv. 在……上面；以上
例句 Example
I need to above this right now.
我现在需要prep./ad在……上面这件事。
#5
absolutely /ˈæbsəluːtli/
Lv.1 adv. 绝对地；完全地
例句 Example
I need to absolutely this right now.
我现在需要ad绝对地这件事。
#6
across /əˈkrɒs/
Lv.1 prep./adv. 穿越；对面；横跨
例句 Example
I need to across this right now.
我现在需要prep./ad穿越这件事。
#7
actor /ˈæktər/
Lv.1 n. 演员；行动者
例句 Example
I need to actor this right now.
我现在需要演员这件事。
#8
actually /ˈæktʃuəli/
Lv.1 adv. 实际上；竟然；真的
例句 Example
I need to actually this right now.
我现在需要ad实际上这件事。
#9
advice /ədˈvaɪs/
Lv.1 n. 建议；忠告
例句 Example
I want to learn how to use 'advice' correctly.
我想学习如何正确使用'advice'。
#10
afraid /əˈfreɪd/
Lv.1 adj. 害怕的；担心的
例句 Example
I need to afraid this right now.
我现在需要害怕的这件事。
